# Balatro 牌型系统拆解

## 核心分层

Balatro 的清晰感来自严格分层：

- 普通牌：只提供点数、花色、基础筹码。
- 牌型：识别普通牌组合，提供固定基础分和倍率。
- Joker：修改规则、放大收益、制造构筑方向。
- 消耗牌：临时干预牌组、牌型等级或单卡属性。

我们的 MVP 直接复用这套分层：

- 基础牌 = 花色 / 相位 + 点数 + 压力。
- 牌型 = 标准扑克牌型。
- 红温 Joker = 规则修改器。

## 牌型表

当前实现使用 Balatro 初始牌型数值：

| 牌型 | 条件 | 基础分 | 倍率 |
| --- | --- | ---: | ---: |
| 高牌 | 其他任意出牌 | 5 | x1 |
| 对子 | 2 张同点数 | 10 | x2 |
| 两对 | 两组对子 | 20 | x2 |
| 三条 | 3 张同点数 | 30 | x3 |
| 顺子 | 5 张连续点数 | 30 | x4 |
| 同花 | 5 张同花色 / 相位 | 35 | x4 |
| 葫芦 | 三条 + 对子 | 40 | x4 |
| 四条 | 4 张同点数 | 60 | x7 |
| 同花顺 | 同时满足顺子和同花 | 100 | x8 |

## 结算顺序

1. 识别牌型。
2. 加上牌型基础分。
3. 乘上牌型倍率。
4. 逐张结算基础牌点数和压力。
5. 红温 Joker 根据压力继续改写倍率、基础分和重复触发。

## MVP 注意点

当前基础牌已经使用 13 档点数：2-10 / J / Q / K / A。

Web MVP 现在使用 52 张标准牌组和抽牌堆 / 弃牌堆，因此顺子概率已经从早期 5 档模板回到更接近标准扑克牌的曲线。后续平衡应优先看红温压力、目标曲线和 Joker 强度，而不是继续扩大基础牌复杂度。
