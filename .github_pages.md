# GitHub Pages 发布（可直接打开玩）

当前仓库使用 `.github/workflows/deploy-pages.yml` 自动发布。

1. 在仓库 Settings → Pages 中选择 GitHub Actions。
2. 推送到 `main` 后等待部署完成。
3. 可玩入口为：
   - `https://<你的用户名>.github.io/<仓库名>/web/`

Pages artifact 会包含：

- `web/`
- `art/`
- `public/`

不要只发布 `/web` 目录，否则 `../art/` 和 `../public/assets/` 下的运行时图片会缺失。
