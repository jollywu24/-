## 视觉方向

当前 UI 方向：

- 黑暗赌桌
- 旧纸牌
- 暗金边框
- 暗红高亮
- 红眼母题
- 克制发光
- 脏、旧、危险的材质，而不是干净霓虹

避免：

- 赛博朋克 HUD 风格
- 明亮彩虹效果
- 彩色噪点
- 常驻粒子
- 过亮的赌鬼卡
- 多处重复展示同一信息

## 强度规则

默认 UI 应该黑暗、冷静。

强发光只留给：

- 选中的手牌
- 当前正在结算的牌
- 红眼状态
- 激活的红眼赌注弹窗
- 已触发的赌鬼卡

不要给静态赌鬼卡、未激活锁定入口或背景装饰使用强发光。

## 字体

- 只使用系统可用字体。
- 标题和按钮可以使用偏衬线感的中文显示字体。
- 按钮和卡牌文字保持短。
- 不使用外部字体。

## 形状与布局

- 主画布为 `1600 x 900`。
- 卡牌旧纸质感只用 CSS 渐变制作。
- 主面板使用黑暗框架和微弱暗金边框。
- 卡牌、按钮和锁定入口可以使用切角。
- 避免大圆角 SaaS 风格卡片。

## 资源约束

当前 UI 规格禁止：

- 外部图片
- 外部字体
- 画布渲染
- 外部框架

允许：

- DOM
- CSS 渐变
- CSS 阴影
- CSS 伪元素
- Unicode 花色符号
